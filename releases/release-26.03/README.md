### TL;DR

The 26.03 release cycle is proposed as follows

**Note:** Following the successful prioritization of our graduation milestone, the standard release cycle has now been reinstated. This shorter-term release is crucial for integrating the significant updates and feature work contributed by all working groups over the recent months. Shorter release cycle due to first release to follow CalVer.

- **Monday, Jan 26th 2026** - Release team is formed
- **Monday, Jan 26th 2026** - Release cycle begins
- **Monday, Mar 2nd 2026** - Feature freeze, manifests sync starts, RC.1 is cut
- **Monday, Mar 16th 2026** - Distribution testing ends, pre-release bug fixing stage starts
- **Monday, Mar 20 2026** - Pre-release bug fixing stage ends
- **Monday, Mar 20 2026** - Docs update ends
- **Monday, Mar 20 2026** - Kubeflow v26.03 Released
